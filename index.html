Survival analysis with multiple causes of death

In our recent paper in Epidemiology (2017; 28(1): 12-19) we proposed the first statistical model for survival data that acknowledges that death may be caused by several diseases acting together by considering all the diseases mentioned on the death certificate. The model enables the estimation of both burden-of-disease indicators and epidemiological association parameters in a way that better reflects the contribution of each disease to mortality. This is particularly important for chronic and degenerative diseases.

  • Paper in Epidemiology, eAppendix and video abstract
  • R package ‘survMCOD’: Scroll down the page in the link to learn how to install and use this R package to fit Cox models for multiple-cause mortality data using the model proposed in our paper.

Time-dependent covariates in survival analysis

Time-dependent covariates are typically incoporated in hazard rate models using a Last Observation Carried Forward (LOCF) approach, i.e. using the last available measurement as the value of the covariate at each event time. With measurement error or missing data this approach is biased. We developed a two-stage joint modelling approach to handle these issues when dealing with multiple continuous markers (paper under revision).

  • R package ‘survtd’: Use this R package to apply the usual LOCF or our approach to incorporate multiple time-dependent covariates in the Cox and additive hazard models.

Measuring socioeconomic inequalities in health

In a paper in Epidemiology (2015; 26(4):518-527) we studied the definitions and estimation of the Relative Index of Inequality (RII) and the Slope Index of Inequality (SII), two indices used to measure relative and absolute gradients in health across the socioeconomic scale. We specifically proposed methods to estimate these indices with individual or aggreated time-to-event data.
